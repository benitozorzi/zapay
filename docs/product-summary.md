# Resumo do produto

Zapay é um app embedded para Shopify Brasil voltado à recuperação de receita via WhatsApp.

## Tese central
Transformar mensagens de recuperação em leitura clara de ROI.

## Fontes de oportunidade
1. checkout abandonado
2. pedido com pagamento pendente

## MVP
- capturar oportunidades da Shopify
- salvar no banco
- exibir dashboard com KPIs
- abrir WhatsApp Web com mensagem montada
- registrar tentativas
- rodar sync manual
- reconciliar conversões depois

## KPIs esperados
- total capturado
- total elegível
- total enviado
- total convertido
- valor líquido recuperado
